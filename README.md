# 🛍️ Brand — E-Commerce Website

A fully responsive B2B e-commerce website built as part of the **DevelopersHub Corporation Frontend Development Internship**. The project is a pixel-perfect clone of a Figma design, replicating an Alibaba-style marketplace called **Brand**.

---

## 🔗 Live Demo

> Open `https://israsif.github.io/ecommerce-frontend-design/` in your browser — no server required.

---

## 📸 Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero banner, deals, category sections, supplier form, recommended items |
| Product Listing | `listing.html` | Filter sidebar, list/grid view toggle, pagination |
| Product Detail | `product.html` | Image gallery, price tiers, specs, tabs, seller card |
| Cart | `cart.html` | Cart items, save for later, coupon code, order summary |

---

## ✨ Features

- ✅ Sticky header with search bar and category filter
- ✅ Hero section with live banner image and category sidebar
- ✅ Deals & Offers section with live countdown timer
- ✅ Home & Outdoor and Consumer Electronics category rows
- ✅ Send Quote form for supplier inquiries
- ✅ Recommended items grid (10 products)
- ✅ Extra services section with background images
- ✅ Suppliers by region with country flags
- ✅ Newsletter subscription form
- ✅ Product listing with list/grid view toggle
- ✅ Filter sidebar with category, brand, price, condition, ratings
- ✅ Active filter tags with remove functionality
- ✅ Product detail with image gallery thumbnails
- ✅ Price tiers table and product spec table
- ✅ Product tabs (Description, Reviews, Shipping, About Seller)
- ✅ Shopping cart with localStorage persistence
- ✅ Add to Cart / Save for Later / Move to Cart
- ✅ Coupon code system (try: `SAVE10`)
- ✅ Order summary with discount and tax calculation
- ✅ Fully responsive — mobile, tablet, desktop
- ✅ Font Awesome icons throughout

## 🛠️ Built With

| Technology | Usage |
|------------|-------|
| **HTML5** | Page structure and semantics |
| **CSS3** | Styling, Grid, Flexbox, responsive design |
| **Vanilla JavaScript** | Cart logic, countdown timer, view toggle, tabs |
| **localStorage** | Cart and saved items persistence |
| **Font Awesome 6** | Icons throughout the UI |
| **Google Fonts (Inter)** | Primary font family |

---

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|------------|--------|
| `> 1024px` | Full desktop layout |
| `≤ 1024px` | Adjusted grid columns |
| `≤ 960px` | Stacked hero, hidden sidebar |
| `≤ 768px` | 2-column grids, mobile nav |
| `≤ 600px` | Single column, compact header |
| `≤ 400px` | Full mobile optimization |
